# Project Lucid

_Follow updates on [Tumblr](http://spiralwise.tumblr.com/)._

Lucid Engine is a openGL based graphics engine.

Using `Glew (1.10.0)` (july 2013), `GLFW (3.0.3)` (september 2013) and `GLM (0.9.4.3)`.
